задача сверстать Landing страницу из файла index.psd с учетов затраченого времени.
Допускается использование CSS фреймворков, дополнительных JS библиотек, любых Open source проектов для оптимизации работы.

Основные требования:

1. Обеспечение выразительно и понятного кода
2. Комментирование HTML/CSS/JS.
3. Отказоустойчивость интерфейса (Graceful Degradation).
4. Респонсив / Мобильная оптимизация.
5. Семантика / W3C спецификации.
6. Использование препроцессоров SASS или LESS.
7. Cross browser compatibility (Кроссбраузерность).

Дополнительные требования(не обязательные):

1. Использование Gulp / Grunt / webpack.
2. Использование Post процесинга - Autoprefixer, PostCSS, Sourcemaps, etc.
3. Примининеие Анимаций, красивых визуальных эффектов.
4. Использование SVG / Canvas.
5. Оптимизация страницы для мобильных.
